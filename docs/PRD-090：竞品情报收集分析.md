# PRD-090：竞品情报收集分析（适配Trae开发）

## 1. 概述
- **目标用户**：市场总监、战略分析师、运营。
- **核心价值**：自动抓取竞品动态（价格、活动、新品、评价），生成分析报告，辅助决策。
- **技术目标**：基于爬虫 + NLP，定时采集公开数据，结构化存储，支持可视化对比。

## 2. 用户故事
- 作为**市场总监**，我想看竞品“XX乐园”最近一个月的新品发布和用户评价，分析其优劣势。
