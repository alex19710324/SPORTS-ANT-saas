# PRD-088：门店证照管理（适配Trae开发）

## 1. 概述
- **目标用户**：行政专员、店长、法务。
- **核心价值**：集中管理门店各类证照（营业执照、消防许可证、卫生许可证等），设置到期提醒，确保合规经营。
- **技术目标**：基于文件存储 + 定时任务，支持证照上传、版本管理、到期自动提醒。

## 2. 用户故事
- 作为**行政专员**，我将门店营业执照扫描件上传，设置有效期，系统在到期前30天自动提醒我续期。

## 3. 功能列表（按优先级）

### 3.1 证照档案（P0）

| ID | 功能名称 | 描述 | 技术实现要点 | 验收标准 |
|----|----------|------|--------------|----------|
| LC01 | 证照录入 | 上传证照图片/PDF，填写名称、编号、发证机关、有效期。 | 文件服务 | 上传成功 |
| LC02 | 证照分类 | 支持分类（营业执照、许可证、资质证书）。 | 标签 | 分类清晰 |
