# PRD-063：广告位与媒介资源管理（适配Trae开发）

## 1. 概述
- **目标用户**：市场媒介、招商经理、财务、合作客户。
- **核心价值**：管理门店内外广告位（大屏、灯箱、展位）的租赁、排期、费用、上下刊，实现广告资源价值最大化。
- **技术目标**：基于Spring Boot + 排期引擎，集成财务底座、合同管理，支持资源日历、在线预订、自动上下刊提醒。

## 2. 用户故事
- 作为**媒介经理**，我配置门店广告位资源（位置、尺寸、价格），生成排期日历。
- 作为**客户**（合作方），我想租赁入口灯箱一周，在线提交申请，支付费用，系统自动排期。

## 3. 功能列表（按优先级）

### 3.1 广告位管理（P0）

| ID | 功能名称 | 描述 | 技术实现要点 | 验收标准 |
|----|----------|------|--------------|----------|
| AD01 | 资源录入 | 录入广告位编号、位置、类型（大屏/灯箱/展位）、尺寸、价格。 | 接口：`/api/ad/resources` | 信息准确 |
| AD02 | 资源日历 | 每个广告位生成排期日历，展示已预订/空闲时段。 | 日历组件 | 实时同步 |

### 3.2 租赁管理（P0）

| ID | 功能名称 | 描述 | 技术实现要点 | 验收标准 |
|----|----------|------|--------------|----------|
| AD03 | 在线预订 | 客户选择时段，提交预订申请，上传素材。 | 接口：`POST /api/ad/book` | 预订成功 |
| AD04 | 合同审批 | 生成租赁合同，双方电子签章，审批后自动锁定时段。 | 电子签章 | 流程闭环 |
| AD05 | 上下刊提醒 | 排期开始/结束前自动提醒执行上下刊。 | 消息通知 | 提醒及时 |

### 3.3 财务结算（P1）

| ID | 功能名称 | 描述 | 技术实现要点 | 验收标准 |
|----|----------|------|--------------|----------|
| AD06 | 费用核算 | 根据合同金额、服务周期自动生成应收款，推送财务。 | 财务底座 | 账务准确 |

## 4. 依赖模块
