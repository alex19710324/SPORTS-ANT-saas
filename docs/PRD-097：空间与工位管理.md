# PRD-097：空间与工位管理（适配Trae开发）

## 1. 概述
- **目标用户**：行政、员工（预约工位）。
- **核心价值**：管理办公区工位/会议室，支持员工在线预约、扫码签到、释放，提升空间利用率。
- **技术目标**：结合地图可视化 + 预约引擎，支持灵活预约规则。

## 2. 用户故事
- 作为**员工**，我想预约明天靠窗的工位，系统显示空闲工位图，我点击预订。

## 3. 功能列表（按优先级）

### 3.1 空间地图（P0）

| ID | 功能名称 | 描述 | 技术实现要点 | 验收标准 |
|----|----------|------|--------------|----------|
| SP01 | 楼层平面图 | 上传平面图，标记工位/会议室位置、编号。 | 拖拽标记 | 清晰 |
| SP02 | 状态实时 | 展示各工位状态（空闲/占用/维护）。 | WebSocket | 实时 |

### 3.2 预约管理（P0）

| ID | 功能名称 | 描述 | 技术实现要点 | 验收标准 |
|----|----------|------|--------------|----------|
| SP03 | 预约规则 | 设置可预约时间段、最大时长、提前预约天数。 | 规则引擎 | 生效 |
| SP04 | 扫码签到 | 员工到达后扫码签到，自动占用工位。 | 接口：`POST /api/space/checkin` | 签到成功 |

### 3.3 统计报表（P2）

| ID | 功能名称 | 描述 | 技术实现要点 | 验收标准 |
|----|----------|------|--------------|----------|
