# PRD-026：员工培训与发展管理（适配Trae开发）

## 1. 概述
- **目标用户**：HR培训专员、部门经理、店长、全体员工（需要接受培训）。
- **核心价值**：构建系统化的员工培训与发展体系，涵盖培训计划制定、课程管理、在线学习、线下培训签到、考核评估、资质认证、晋升关联，提升员工岗位技能与综合素质，支撑人才梯队建设。
- **技术目标**：基于Spring Boot + Vue3微服务架构，集成学习管理引擎、考试引擎、证书管理、与人事绩效模块联动，支持多终端（PC、移动端）学习，培训覆盖率100%，考核结果自动同步至绩效系统。

## 2. 用户故事
- 作为**HR培训专员**，我想要根据岗位需求制定年度/月度培训计划，上传培训材料（视频、PPT、文档），发布培训通知，并跟踪员工学习进度。
- 作为**新员工**，我想要在入职后自动收到岗位必修课清单，在线学习课程并参加考核，获得上岗资质。
- 作为**导玩员**，我想要学习新设备玩法培训课程，通过考核后获得该设备的操作认证。
- 作为**店长**，我想要查看本店员工培训完成率、考核通过率，了解团队技能短板，安排针对性培训。
- 作为**晋升评审**，我想要查看候选人的培训记录和认证资质，作为晋升依据。

## 4. 技术架构
- **前端**：Vue3（PC端）+ 小程序（移动端）。
- **后端**：Spring Boot + 学习管理微服务。
- **数据**：MySQL（课程、计划、记录）、Redis（缓存）、Elasticsearch（搜索）。
- **底座依赖**：
  - 通讯社媒底座：通知推送。
  - 数据中台：报表分析。
  - 人事系统：员工档案、晋升联动。

## 5. 接口设计示例
- **员工学习记录上报**：
  - `POST /api/training/records`
  - 请求：
    ```json
    {
      "employeeId": 123,
      "courseId": 5001,
      "progress": 80, // 百分比
      "status": "in_progress",
      "lastPosition": "00:15:30"
    }
    ```

- **获取岗位学习地图**：
  - `GET /api/training/map?positionId=2001`
  - 响应：
    ```json
    {
      "positionName": "导玩员",
      "requiredCourses": [
        { "courseId": 5001, "name": "新人入职培训", "completed": true },
        { "courseId": 5002, "name": "设备操作基础", "completed": false }
      ],
      "certifications": [
        { "certId": 3001, "name": "安全员认证", "expired": "2026-01-01" }
      ]
    }
    ```

## 6. 验收标准
- 课程创建与发布流程顺畅。
- 在线学习进度记录准确。
- 签到成功率≥99%。
- 考试成绩与证书自动关联。
- 报表数据与学习记录一致。

## 7. 依赖模块
- **通讯社媒底座**：通知推送。
- **数据中台**：培训效果分析。
- **人事系统**：岗位定义、晋升联动。
- **员工工作台**：学习入口。
