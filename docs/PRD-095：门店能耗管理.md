# PRD-095：门店能耗管理（适配Trae开发）

## 1. 概述
- **目标用户**：店长、后勤、财务。
- **核心价值**：实时监测门店水、电、气能耗，分析能耗成本，发现异常用能，提出节能建议，降低运营成本。
- **技术目标**：对接智能电表/水表，数据采集分析，支持预警、报表。

## 2. 用户故事
- 作为**店长**，我看到昨日电费突增，系统提示“空调异常开启”，我及时调整。

## 3. 功能列表（按优先级）

### 3.1 能耗采集（P0）

| ID | 功能名称 | 描述 | 技术实现要点 | 验收标准 |
|----|----------|------|--------------|----------|
| EG01 | 仪表对接 | 对接智能电表、水表、燃气表，实时读取数据。 | Modbus/MQTT | 数据准确 |
| EG02 | 手工录入 | 支持手工录入（无智能表）。 | 接口：`POST /api/energy/record` | 录入成功 |
