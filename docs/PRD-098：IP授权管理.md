# PRD-098：IP授权管理（适配Trae开发）

## 1. 概述
- **目标用户**：品牌经理、法务、产品开发。
- **核心价值**：管理IP授权合作（如联名盲盒），包括授权期限、分成比例、素材使用、销售数据上报，确保合规。
- **技术目标**：基于合同管理 + 销售数据，支持授权到期提醒、分成计算。

## 2. 用户故事
- 作为**品牌经理**，我与某IP方签订联名授权合同，录入系统，系统根据盲盒销售额自动计算分成。

## 3. 功能列表（按优先级）

### 3.1 授权合同（P0）

| ID | 功能名称 | 描述 | 技术实现要点 | 验收标准 |
|----|----------|------|--------------|----------|
| IP01 | 合同录入 | 记录IP名称、授权方、授权期限、分成比例、最低保证金。 | 接口：`/api/ip/contracts` | 录入成功 |
| IP02 | 素材管理 | 关联IP授权素材（图库、设计稿）。 | 素材库 | 关联 |

### 3.2 分成计算（P1）

| ID | 功能名称 | 描述 | 技术实现要点 | 验收标准 |
|----|----------|------|--------------|----------|
| IP03 | 销售数据 | 自动获取联名商品销售数据，按规则计算分成。 | 财务底座 | 准确 |
| IP04 | 分成报表 | 生成授权方分成报表，支持导出。 | 报表 | 清晰 |

### 3.3 到期提醒（P0）

| ID | 功能名称 | 描述 | 技术实现要点 | 验收标准 |
|----|----------|------|--------------|----------|
| IP05 | 提醒 | 授权到期前自动提醒续约。 | 消息中心 | 及时 |

## 4. 依赖模块
- 财务底座（PRD-009）
- 素材库（PRD-089）
