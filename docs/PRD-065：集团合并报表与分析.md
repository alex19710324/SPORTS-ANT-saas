# PRD-065：集团合并报表与分析（适配Trae开发）

## 1. 概述
- **目标用户**：总部财务总监、CFO、董事会成员、区域总经理。
- **核心价值**：自动汇聚各门店、各子公司的财务与经营数据，生成集团合并报表（合并利润表、资产负债表、现金流量表），支持多维度分析（区域、业态、时间），为高层决策提供精准数据支撑。
- **技术目标**：基于数据中台+财务底座，支持多会计准则（国内准则、IFRS）、自动抵消内部交易、合并报表一键生成，报表生成时间≤5分钟。

## 2. 用户故事
- 作为**CFO**，我需要每月5日看到集团合并报表，包括各区域营收占比、利润贡献、现金流健康度，并能下钻查看异常门店。
- 作为**审计师**，我需要追溯合并报表中的内部交易抵消明细，确保数据合规。
