# PRD-093：客流热力分析（适配Trae开发）

## 1. 概述
- **目标用户**：运营总监、店长、市场专员。
- **核心价值**：通过WiFi探针/摄像头/票务系统采集客流数据，生成热力图、时段分析、游逛深度，辅助运营决策。
- **技术目标**：基于大数据分析 + GIS地图，支持实时/历史客流分析。

## 2. 用户故事
- 作为**店长**，我想看今天哪个区域人流最密集，优化设备摆放和人员排班。
