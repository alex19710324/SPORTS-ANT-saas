# PRD-045：门店官网/品牌展示网站（PC端网页1·适配Trae开发）

## 1. 概述
- **目标用户**：潜在消费者、到店前浏览顾客、门店品牌关注者。
- **核心价值**：打造品牌形象窗口，提供门店服务介绍、项目展示、预约入口、活动资讯、会员权益说明，吸引顾客到店体验，提升品牌认知。
- **技术目标**：基于Vue3 + Nuxt.js（SSR）开发响应式官网，集成语言底座（多语言）、预约系统、内容管理，支持SEO优化，页面加载速度≤2秒。

## 2. 用户故事
- 作为**潜在顾客**，我在搜索引擎搜索“运动蚂蚁”，进入官网首页，看到门店环境、热门项目、优惠活动，点击“立即预约”跳转到小程序预约页面。
- 作为**门店店长**，我可以在后台更新官网的Banner、活动信息、项目介绍，无需开发介入。
- 作为**品牌运营**，我想在官网发布品牌故事、IP合作新闻，提升品牌形象。

## 3. 功能列表（按优先级）

### 3.1 官网前端展示（P0）

| ID | 功能名称 | 描述 | 技术实现要点 | 验收标准 |
|----|----------|------|--------------|----------|
| WS01 | 首页 | 展示品牌Slogan、环境图、热门项目、最新活动、门店列表、预约入口。 | Nuxt.js + 动态组件 | 加载≤2秒 |
| WS02 | 项目详情页 | 分类展示各运动/潮玩项目（VR、蹦床、保龄球等），含图文介绍、价格、玩法视频。 | 接口：`GET /api/projects` | 内容准确 |
| WS03 | 活动资讯页 | 展示近期活动、赛事、促销，支持按时间筛选。 | 接口：`GET /api/activities` | 更新及时 |
| WS04 | 门店列表页 | 地图模式/列表模式展示全国门店，点击查看门店详情（地址、电话、营业时间）。 | 地图API集成 | 定位准确 |
| WS05 | 会员权益页 | 介绍会员等级、积分规则、专属权益，引导注册。 | 静态页面 | 信息清晰 |

## 4. 技术架构
- **前端**：Nuxt.js（SSR）+ Vuetify。
- **后端**：Spring Boot + CMS微服务。
- **数据**：MySQL（内容）、Redis（缓存）。
- **底座依赖**：语言底座、预约系统。

## 5. 接口设计示例
- **获取首页Banner**：
  - `GET /api/cms/banners?position=home`
  - 响应：Banner列表（图片、链接）。

## 6. 验收标准
- 首页加载≤2秒。
- 后台内容更新10秒内生效。
- 多语言切换流畅。
